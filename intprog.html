Introduction to Programming (INTPROG)

This unit covers the very basic concepts of computer programming. It begins with Python, where programming techniques are taught up to creating an interactive graphics window. These skills are then transitioned into Java.

Python Coursework

This courswork involved making various "patches" in the graphics window. They had to be designed and ordered based on the last three digits of our student number (with 857172 I had "bottom-left triangle" shape, zig-zags for the first shape and radial lines for the second).

The user would be asked for input on the size (either 5x5, 7x7, 9x9 or 11x11 patches) and then on the three different colours the patches should be generated with.
